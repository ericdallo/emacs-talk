~C~ - Control
~S~ - Shift
~SPC~ - Space
~RET~ - Return (Óåõ)

* Modes

*** Major mode

- Only 1 major mode per buffer

- Examples: ~dart-mode~, ~java-mode~, ~clojure-mode~

*** Minor modes

- Many minor modes per buffer

- Examples: ~cider-mode~, ~evil-mode~, ~lsp-mode~
